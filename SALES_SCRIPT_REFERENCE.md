# Sales Script Reference (Internal Only)

**Este documento es solo para referencia interna. NO debe aparecer en la página web.**

Este script guía el copy y la estructura de conversión de la landing page, pero no se muestra directamente al usuario.

## Estructura de Persuasión

### Paso 01 — Lo que no ve
**Concepto**: Crear conciencia del problema oculto
- Los pacientes inactivos representan pérdida de ingresos (150€-800€ cada uno)
- Nadie les recordó que existen
- **Aplicado en**: Hero + Auditoría de Valor

### Paso 02 — La hemorragia invisible
**Concepto**: Cuantificar el costo administrativo
- Recepción invierte horas en tareas repetitivas
- Costo equivalente a salario completo
- **Aplicado en**: Auditoría de Valor (sistemas de automatización)

### Paso 03 — Cuánto le cuesta
**Concepto**: Números concretos de pérdida
- 20% no-shows = 3 citas/día perdidas
- 360€ diarios = 7.000€+ mensuales
- **Aplicado en**: Estadísticas + Planes (ahorro mensual destacado)

### Paso 04 — El cambio
**Concepto**: Solución = infraestructura, no marketing
- Automatización 24/7 (responder, agendar, confirmar, recordar, reactivar)
- Medible y automático
- **Aplicado en**: Auditoría de Valor + Planes

### Paso 05 — Quiénes somos
**Concepto**: Credibilidad por instalación profesional
- Se instala una vez, se mantiene periódicamente
- Produce resultados cada día
- **Aplicado en**: Sección Instalación

### Paso 06 — La propuesta
**Concepto**: Eliminar fricción de decisión
- 7 días gratis, sin riesgo
- Ver resultados con datos reales
- Desinstalación sin coste si no convence
- **Aplicado en**: Sección Prueba Gratuita + CTA Final

## Principios de Conversión Implementados

1. **Jerarquía visual extrema**: Números grandes para impacto emocional
2. **Urgencia implícita**: "Cada cita perdida cuesta..." + "Mientras otros gestionan..."
3. **Prueba social**: Beneficios cuantificables (450€/mes, 1.501€/mes ahorro)
4. **Reducción de fricción**: "7 días gratis", "Sin compromiso", "15 minutos"
5. **Cierre múltiple**: 3 CTAs estratégicamente colocados (Hero, Planes, Final)
6. **Confianza**: Instalación profesional + formación + soporte

## Elementos de Página que Refuerzan el Script

- **Hero**: Establece el problema ("Menos gestión. Más ingresos")
- **Estadísticas**: Datos que resuenan con el dolor (73%, 15-30%, 24/7)
- **Auditoría de Valor**: Cuantifica cada solución (450€/mes, 500€/mes, etc.)
- **Planes**: Muestra ROI inmediato (59€/mes = 1,97€/día)
- **Prueba Gratuita**: Elimina riesgo de decisión
- **CTA Final**: Cierre con urgencia suave ("¿Qué día empezamos?")
