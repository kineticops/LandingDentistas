# Kinetic Ops — Landing Page

Landing page para Kinetic Ops, sistema de IA para clínicas dentales.

## Archivos

- `index.html` — Landing page completa (HTML + CSS + JS en un solo archivo). Listo para GitHub Pages o cualquier hosting estático.

## Despliegue rápido con GitHub Pages

1. Sube `index.html` a la raíz de tu repositorio
2. Ve a **Settings → Pages**
3. En Source, selecciona **Deploy from a branch → main → / (root)**
4. Tu landing estará live en `https://kineticops.github.io/LandingDentistas/`

## Edición

El archivo es un HTML puro autocontenido. Para editarlo:

- **Textos**: Busca y reemplaza directamente en el HTML
- **Colores**: Modifica las variables CSS en `:root` al inicio del archivo
- **Precios**: Busca los valores en € dentro del HTML
- **Email de contacto**: Busca `kineticopss@gmail.com` y cámbialo si necesitas

## Stack

- HTML5 + CSS3 + Vanilla JS
- Google Fonts: Plus Jakarta Sans
- Sin dependencias externas
- Sin build necesario
